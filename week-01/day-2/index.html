Wiki definitions

Computers and the Internet

NOT
It is logic operator based on boolean algebra. Have two operators true, false and simply change value (negation).
AND
This operator has two inputs and one output. Logical terms true or false, if both inputs are true, there will be true on output. In another case will be false.
OR
This operator has two inputs and one output. Logical terms true or false, if one input is true, there will be true on output. Only if both inputs are false, there will be false on output.
XOR
This operator is like OR, but there is one difference. If both inputs are true, there will be false on output.
Binary code
A binary code represents text, computer processor instructions, or any other data using a two-symbol system (0,1).
Bit1
It represents basic binary digit, 1 or 0.
Byte
A byte is 8 bits.
Character Code (ASCII)
The American Standard Code for information interchange.
CPU
Central processing unit
Control Unit
CU is in the CPU. It tells the computer's memory, arithmetic/logic unit and input and output devices how to respond to the instructions that have been sent to the processor.
ALU
Arithmetic and Logic Unit
Instruction Set
It is a set of instruction to do between CPU and RAM.
Clock
It is a particular wire that turns on and off a steady rate to help keep everything sync.
RAM
Random access memory
Memory Address
Is a unique identifier used by a device or CPU for data tracking.
Bandwith
The maximum transmission capacity of a device.
Bitrate
It is measure bandwidth, which is the number of bits that we can actually send overa given period of time.
Packet
The packet is your data you want to send or receive through the network.
Router
Is a networking device that forwards data packets between computer networks.
Internet protocol (IP)
Is a protocol by which data (packets) is sent from one computer to another on the Internet.
Transmission Control Protocol (TCP)
Manages the sending and receiving of all your data as packets.
Internet Service Provider (isp)
It is usually a company which connected me with all other devices in the world through hundreds of thousands of networks.
IP Address
They are unique numbers to each device on the network
Domain Name System (DNS)
Associates names with the corresponding addresses.
Encryption
It is the form of scrambling or changing a message to hide it.
Security Protocols (https)
This protocol is using public key encryption to exchange data securely with the website you are on.
Virus
Is an executable program harms a user and their computers.
DDoS
Distributed denial of service, when hackers overwhelm a website with too many requests.
Phising
Is to send large amounts of spam email in an attempt to trick people into sharing sensitive personal information.
